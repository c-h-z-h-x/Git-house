# 概率统计类

清华开设的此类课程较多，此处只含最常见的三门：

* **概率论与数理统计**：此为大部分需要学习概率统计类课程的必修课，开课老师众多，教学体系差异，难度差异大，考试现在为期末50分统考50分各自出卷。
* **概率论与随机过程**：此课程一般与**概率论与数理统计**并列，作为部分院系二选一的必修课。难度与概统接近，其中秋季学期由yj老师开设，难度较低，随统一致。**(此课程不是电子系的概率论与随机过程)**
* **随机数学与统计**：此课程一般与前两者并列，作为部分院系三选一的必修课（此外，是自动化系的必修课），5学分，内容偏多，但难度一般。

理论上来说，这三门课就是概率论、数理统计、随机过程的不同组合。
